#  Python Project: Time Tracker for Personal Task Management

### Software Development, Process Automation, and Data Handling using Python

This project is a functional, command-line utility built entirely in Python designed to help users track and manage time spent on various activities. It demonstrates foundational skills in building robust, data-driven applications from the ground up.

## Key Project Features & Objectives:
* **Automation:** Developed a complete solution to automate the process of recording task start/stop times and calculating total session duration.
* **Data Persistence:** Records activity data and durations to ensure data integrity across sessions.
* **Core Functionality:**
    * Start tracking time for an activity.
    * Stop tracking time and record the duration of the activity.
    * Display a summary of all recorded activities and their total durations.
* **Data Processing:** Utilized Python to clean, aggregate, and calculate key time metrics for summary reporting.

## How to Run the Project

This section guides users on setting up and running the application using PyCharm.

### 1. Open the Project in PyCharm
* Clone the repository or download the project folder.
* Open the project in PyCharm.

### 2. Run the Script
* Locate the `603_project vema.py` file in the project directory.
* Right-click on the file and select **Run '603_project vema'** or press `Shift + F10`.

### 3. Interacting with the Application
Follow the on-screen menu to manage your tasks.
* Start an activity by entering its name.
* Stop the activity to save the time spent.
* Display the summary to review all recorded sessions.
* Select option **4** from the menu to exit the application.
